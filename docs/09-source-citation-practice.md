# Source Citation Practice for Local Notes

When using a local document assistant, source references help users verify the answer. The assistant should show which document chunks were used and provide enough metadata to locate the original note.

## Good Practice

Use document titles, categories, IDs, chunk numbers, and short previews. For formal academic work, the user should still check the original source and apply the required citation style manually.

## Avoiding Unsupported Claims

If a statement is not present in the retrieved context, the assistant should say that the information is not available in the local knowledge base. This is especially important for deadlines, grading rules, statistics, and citations.
