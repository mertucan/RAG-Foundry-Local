# Academic Note-Taking Strategy

Good notes help retrieval systems and human readers. Notes should use descriptive headings, short sections, clear terminology, and consistent metadata. Markdown is practical because headings and lists are easy to parse.

## Recommended Format

Each note should include a title, category, source or course context, and a short summary. Important definitions should be written in complete sentences. Lists are useful for procedures, arguments, comparison points, and revision checklists.

## Preparing Notes for RAG

Avoid very long unstructured paragraphs. Split topics into sections so the chunker can keep related information together. If a PDF is used, export or summarise the relevant content as markdown or text before ingestion.
